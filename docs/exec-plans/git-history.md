# git-history.md

## 목적

이 문서는 PackUp 프로젝트의 개발 흐름과 변경 이력을 기록한다.

단순 커밋 로그가 아니라  
“왜 변경했는지”를 함께 기록한다.

---

## 규칙

- 기능 단위로 기록
- 커밋 / 이슈 / PR 연결
- 변경 이유 포함

---

## 예시

### 🧩 Feature: 상황 선택 UI

#### 🔹 Issue
- #1 상황 선택 화면 구현

---

#### 🔹 Commit

fix: 긴급 에러 수정
- 상태 업데이트 안되는 문제 해결
- 버튼 클릭 이벤트 누락 수정

feat: 상황 선택 UI 구현
- 여행 / 학교 / 헬스 버튼 UI 추가
- 선택 상태 관리 로직 구현

---

#### 🔹 PR

# PR 제목

상황 선택 UI 구현

---

## 변경 내용

- 상황 선택 버튼 UI 추가
- 상태 관리 로직 구현

---

## 이유

- 사용자 첫 진입 경험 개선
- 빠른 선택 흐름 제공

---

## 확인 사항

- 버튼 클릭 정상 동작
- 상태 정상 변경

---

#### 🔹 Why

- 사용자가 첫 진입 시 바로 선택 가능하도록 하기 위해
- UI 단순화를 위해 버튼 구조 선택

---

#### 🔹 결과

- 상황 선택 가능
- 다음 단계 이동 가능
