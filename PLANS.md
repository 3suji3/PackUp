# PLANS.md

## 1. 목표

PackUp MVP를 안정적으로 구현한다.

핵심 기능:
- 상황 선택
- 체크리스트 자동 생성
- 체크 상태 관리
- 항목 추가
- 체크리스트 여러 개 저장 및 재사용

---

## 2. 작업 원칙

- 한 번에 하나의 기능만 구현
- 각 단계 완료 후 사용자 확인
- 애매하면 질문 후 진행
- MVP 범위 절대 유지

---

## 3. 전체 흐름

Phase 1 → 구조  
Phase 2 → 생성 로직  
Phase 3 → UI  
Phase 4 → 저장  
Phase 5 → UX 개선
